# 弹出框篇

# 设计细节

  1. 触发方式。可设置通过trigger属性设置点击click或者覆盖hover触发弹出框。同时通过点击触发的弹出框，再次点击或者点击页面其它地方，将会关闭该弹出框，但点击弹出框区域不关闭。
  2. 弹出位置。可通过position属性设置弹出位置，有顶部top、底部bottom、左侧left和右侧right。

# 轮播篇

# 设计细节

  1. 轮播分为有缝轮播和无缝轮播，这里采用的是无缝轮播。
  2. 轮播功能主要基于vue过渡&动画实现。
  3. 加上touchStart和touchEnd事件处理，增加对移动端的支持。
  4. 本次父子组件通信使用$parent和$children。
  5. 使用到了updated生命周期来处理轮播的选中更新等操作。
